🎯 COMANDOS PRINCIPAIS:

📝 BUILD - Criar código novo
   agent build "um formulário de login React"
   agent build "página HTML com CSS moderno" -l html
   agent build "componente de abas" -l react -f Tabs.jsx

🔧 FIX - Corrigir bugs
   agent fix "função retorna undefined" -f app.js
   agent fix "erro de layout no mobile" -c "seu código"

👀 REVIEW - Análise de código
   agent review -f componente.jsx
   agent review -f api.ts -d deep

💡 EXPLAIN - Entender código
   agent explain -f funcao.js
   agent explain -c "const sum = (a, b) => a + b"

⚙️  GENERATE - Gerar código específico
   agent generate "validador de email robusto" -l js
   agent generate "hook useAuth" -l react -f useAuth.ts

♻️  REFACTOR - Melhorar código
   agent refactor app.js -d "adicionar TypeScript"
   agent refactor styles.css -d "otimizar performance"

📚 TEACH - Aprender sobre um tema
   agent teach "React Hooks" -l advanced
   agent teach "CSS Grid" -l beginner

📦 CRIAR EXECUTÁVEL .exe (OPCIONAL):

Permite usar "agent" de qualquer lugar no Windows

   npm install -g pkg
   pnpm -C artifacts/frontend-agent-cli run build:exe

Resultado: dist/agent.exe

Adicionar ao PATH:
   1. Copy agent.exe para C:\Program Files\agent\
   2. Windows + X → System → Environment Variables → PATH → New
   3. Cole: C:\Program Files\agent
   4. Reinicie

Depois:
   agent build "seu comando" (de qualquer pasta)
