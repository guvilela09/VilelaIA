⚙️  PRÓXIMOS PASSOS:

1️⃣  CONFIGURAR API KEY (IMPORTANTE!)

   Opção A - Variável do Sistema (recomendado):
   ────────────────────────────────────────
   Antes, obtenha sua chave em: https://console.groq.com
   
   PowerShell (como Admin):
   [Environment]::SetEnvironmentVariable(
     "GROQ_API_KEY", 
     "sua_chave_aqui", 
     "User"
   )
   
   Depois feche e abra PowerShell novamente

   Opção B - Arquivo .env.local:
   ──────────────────────────────
   echo "GROQ_API_KEY=sua_chave_aqui" > .env.local

📚 COMANDOS DISPONÍVEIS:

   agent build <descrição>
   → Criar componente/página novo
   
   agent fix <descrição> -f arquivo.js
   → Corrigir bugs
   
   agent review -f arquivo.js
   → Análise detalhada do código
   
   agent explain -f arquivo.js
   → Entender código
   
   agent generate <descrição> -l react
   → Gerar código em linguagem específica
   
   agent refactor arquivo.js -d "descrição"
   → Melhorar código existente
   
   agent teach <tópico>
   → Aprender sobre um tema (React, CSS, etc)

════════════════════════════════════════════════════════════════

💡 EXEMPLOS RÁPIDOS:

   # React com Hook de Autenticação
   agent build "hook useAuth com login/logout" -l react -f useAuth.ts

   # Corrigir função bugada
   agent fix "retorna undefined" -f processData.js

   # Review profundo
   agent review -f api.ts -d deep

   # Gerar validador
   agent generate "validador de formulário robusto" -l ts

   # Aprender
   agent teach "CSS Grid" -l beginner
